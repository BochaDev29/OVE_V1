## 🧭 Cómo usar esta carpeta

1. **Leé el .md** — explica el concepto en español, con ejemplos del proyecto
2. **Seguí los enlaces** — cada concepto apunta a la línea exacta del código
3. **Modificá algo pequeño** — la mejor forma de aprender es romper algo y entender por qué
4. **Preguntale al asistente** — usá los tokens para preguntas puntuales, no para aprender conceptos base (para eso son estos archivos)

## 🔑 Conceptos previos que ya tenés (Java → Web)

| Java | Equivalente en este proyecto |
|------|------------------------------|
| `class` con métodos | Componente React (función con lógica) |
| `interface` | `interface` en TypeScript (igual concepto) |
| `getter/setter` | Props y estado en React |
| `try/catch` | `try/catch` igual en TypeScript |
| Herencia | Composición de componentes (React prefiere esto) |
| `void main()` | `App.tsx` — punto de entrada |
